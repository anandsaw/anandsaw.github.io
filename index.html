Anand Ashok Sawant

Postdoctoral scholar at University of California, Davis

I am a Postdoc working in the Decal lab at UC Davis with Prof. Premkumar Devanbu. My current research focus is on applying natural language models to new software engineering tasks. My latest work has centered around improving the understandability of decompiled binary code using a pre-trained language model for C. During my PhD I focused on the human aspect of API evolution to understand how and why APIs evolve and its impact on API consumers. Going forward I would like to enhance the API evolution process by devising a semantic aware approach that is able to automatically migrate consumer code from one API to another. Fun fact: the background image of this page is generally related to the area I am currently in.
